My tools

I use Adobe Photoshop (2019 version) as my drawing software of choice. By the way, I'd argue that, not only is it not morally wrong, it is, in fact, every artist's moral imperative to pirate Adobe's Software!.. For legal reasons, this is a joke.

My graphic tablet is Wacom Cintiq Pro DTH-1320.

As for 3D, I use Blender to make the model itself and then later texture it in Substance Painter (circa 2018 version).

For coding and development I use Visual Studio Code.
