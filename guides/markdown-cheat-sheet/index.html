Markdown cheat sheet

Published: 01/28/2025

What is Markdown?

Markdown is a lightweight markup language that lets you add formatting elements to plaintext documents. It's used by a lot of SSGs (like mine, HUGO), but also in places like Discord and Reddit among others. Its main advantage is readability and simple syntax.

Different implementations of Markdown vary a bit between software and platforms. This guide is mostly from my own perspective and includes info about features that I personally use the most.

Basic text formatting

You can separate paragraphs by adding an empty line in between them. You can also add a bit of basic formatting to your text, like shown below.

_italics_ or *italics*

**bold**

~strike-through~ or ~~strike-through~~

`code`

||spoiler|| (Discord only), or >!spoilers!< (Reddit only).

>This is a blockquote. Lorem ipsum dolor sit amet, consectetur adipiscing elit. Aenean malesuada pellentesque erat quis pulvinar. Quisque sed sagittis diam. Phasellus commodo mattis libero, et luctus tellus porta eu.

Headings

You should put a blank line before and after the heading.

# Heading 1

## Heading 2

### Heading 3

#### Heading 4

Links

link

[link text](link URL)

Lists

Ordered list:

  1. first
  2. second
  3. third

Unordered list:

  • first
  • second
  • third

or:

  • first
  • second
  • third

or, sometimes:

  • first
  • second
  • third

Image

![alt text](/path/to/image.jpg)

Horizontal rule (separator)

--- or ***
