Useful software

Also, be sure to check coding resources links and art resources links for more software recommendations!

Last updated: January 30th 2025

For desktop (Windows & Linux)

Joplin — open source note-taking app.

qBittorent — my open source torrent client of choice.

For Android

F-Droid — installable app store of FOSS (Free and Open Source Software) applications for the Android platform.

PipePipe — my current favorite YouTube client for mobile. Has built-in adblock and sponsorblock, doesn’t require a sign-in, but still lets you subscribe to channels locally and lets you customize its layout to make it less distracting. Also, lets you play audio with screen turned off!

Browser extensions (mainly for Firefox in my case)

UBlock Origin — my favorite ad blocker. Also lets you block any section of a website that you find distracting, like home feed.

SponsorBlock — skip annoying YouTube sponsor segments, as well as some other useless filler.

NoScript - lets you manually blacklist or whitelist each individual JavaScript on the website by website basis.

Bitwarden — my current password manager of choice. Has a mobile app too.
