Coding resources links dump

Shameless plug: check out my own guides as well!

Start here

MDN Web Docs — references and documentation on HTML, CSS and JavaScript; useful when you need to look up a particular tag or an attribute.

W3Schools — similar to abovementioned MDN; lots of documentation and tutorials.

Git — version management software.

HUGO static site generator — for making and managing static websites like this one. There are many SSGs out there, HUGO is just the one I personally use.

HUGO learning resources

Make your own website — a simple guide written for the author's 12-year-old daughter on how to actually make your own website.

The only tags you need when first learning HTML — video by Kevin Powell; also a great YouTube channel about CSS!

Specifics

Guide to CSS flexbox and CSS grid by CSS-Tricks; self-explanatory.

CSS grid generator

HTML table generator — DO NOT use this for layouts, please!!!

CSS Logical Properties and Values by CSS-Tricks.

CSS Colors — a cheat sheet from W3Schools.

Setting Height And Width On Images — useful read about embedding images on your site.

W3C CSS Validation Service — same thing, but for CSS; migth give false positive warnings for newly added features sometimes.

Debug CSS Firefox extension — useful for visualizing CSS box model.

UNIX Timestamp Generator — occasionally you might need this for various uses.

Google Fonts — free to use fonts for your website.

Where to host a static site for free

GitHub Pages — my current hosting provider.

Neocities — not a fan, but it's probably the most famous "indie web" platform.

Sidenote: I'm obviously biased, but I consider GitHub Pages to be the clearly superior option to Neocities, tech-wise and service-wise. Neocities whole shtick is that it has a "social media" functionality attached to it, which is both a blessing and a curse (more of a curse, IMO). There are of course, other free hosting providers, but I don't have any experience with them. Oh and Neocities allows porn, so there's that.

Guestbook providers

Atabook — free guestbook provider. No ads and has a plenty of robust moderation options, such as banning the use of VPNs. My current guestbook provider.

Other resources

Resources List for the Personal Web (from 32bit café) — another useful links list.

When using search engines, appending "site:www.reddit.com" or "site:stackoverflow.com" at the end of a search query helps a lot when looking for a solution to a specific coding issue by filtering results to only those sites.

Example: "How to center a <div> site:www.reddit.com/r/webdev".
