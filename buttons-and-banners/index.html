Buttons and banners

About buttons

If you find this website interesting and/or useful and want to share it with others, you can put one of these 90s-style buttons/banners on your own website and link back to me. Please do not hotlink the images and thank you.

What the hell are buttons?!

Buttons are small images, most commonly 88 by 31 pixel in size. They can be static or animated. They are a way for webmasters to exchange links in style and build a community of websites. They were really popular in the 90s and early 2000s and they are making a comeback.

If you have a personal website, you can use one of the button images and create an image-link that links to this website.

88x31

Image:

Code:

HTML (show code)
<a href="https://sqvrltastic.art/" target="_blank" rel="noopener noreferrer">
  <img src="./images/button-sqvrltastic.webp" alt="Sqvrltastic's Homepage">
</a>

Replace the path to the image with your own relative link!

Optional: add CSS to negate browser default zoom level:

CSS (show code)
.small-button {
  /* code to make 88x31 buttons look crispier and negate browser zoom */
  inline-size: 4.2rem;
  aspect-ratio: 88/31;
  transform: scale(1);
}

IMO, don't add image-rendering: pixelated; because it makes it look really weird and ugly on desktops with non-standard zoom.
